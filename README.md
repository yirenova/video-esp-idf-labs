# ESP-IDF Video Labs

This repository contains **ESP-IDF companion projects** for my embedded & IoT tutorial videos.

Each demo is a **Lab** identified by a Case ID (e.g., `E0001`).  
Every lab includes:
- ESP-IDF project (CMake-based)
- Wiring notes / pin mapping
- BOM (parts list)
- Setup steps and troubleshooting notes
